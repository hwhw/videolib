Search syntax help

holiday — full-text search in filenames/paths

holiday* — prefix/truncated search

"beach party" — exact phrase search

tag:test1 — videos with tag "test1"

tag:act* — tags matching "act..." (wildcard)

UNTAGGED — videos with no tags

TAGGED — videos with at least one tag

duration:+1:30 — videos ≥ 90 seconds

duration:-60 — videos < 60 seconds

duration:+1:00:00 — videos ≥ 1 hour

size:+100m — videos ≥ 100 MiB

size:-1g — videos < 1 GiB

size:+500k — videos ≥ 500 KiB

path:videos/2024/* — path glob (case-sensitive, shell rules: * ? [abc])

path:"my videos/vacation/*.mp4" — quoted path with spaces

ipath:*.mkv — case-insensitive path glob

ipath:videos/comedy/* — case-insensitive directory match

tag:action AND tag:comedy — both tags

tag:action OR tag:comedy — either tag

NOT tag:boring — exclude tag

(UNTAGGED OR tag:action) AND NOT tag:boring — grouping

holiday tag:vacation — implicit AND: filename match + tag

"my video" duration:+5:00 size:-500m — combine all filters

ipath:*/comedy/* tag:favorite duration:+10:00 — combine path, tag, duration
